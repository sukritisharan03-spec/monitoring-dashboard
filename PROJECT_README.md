## Objective

The objective of this project is to build a Monitoring Dashboard that provides a centralized view of system status and security-related information. The dashboard helps users monitor system performance, services, logs, and security events through a simple web interface.

---

## Product Description

Monitoring Dashboard is a web-based application developed using React and Vite. It displays system monitoring information such as CPU usage, memory usage, service status, logs, and security-related alerts.

## Commands Used

### Create Project

```bash
npm create vite@latest
```

**Purpose:** Creates a new Vite project.

### Install Dependencies

```bash
npm install
```

**Purpose:** Installs all required project dependencies.

### Run Development Server

```bash
npm run dev
```

**Purpose:** Starts the local development server and hosts the application locally.
